-- Version 1.0 --
-	First release -Needs- testing.

-- Version 1.01 --
-	Added 1 second delay to the server request for rewards. (sometimes the server didn't have the item data ready)

-- Version 1.02 --
-	Fixed showing rewards unless on island expeditions
-	Fixed Battleground Queue taint
-	Fixed ctrl+click and shift+click loot in rewards tab
-	Moved Ettin to Giant classification
-	Fixed a bug that made the window reopen when closed during queue
-	Fixed mismatched mobs's in the database
-	Fixed Ahoy opening during Warfronts

-- Version 1.03 --
-	Moved Obsidian Dragon category to Black Dragon, since they're one and the same when it comes to loot table
-	Added setting - Hide minimap button
-	Added window scale buttons (lets you scale the window up and down, not perfect)
-	Added a window lock button (stops it from being movable)
-	Added Chat Commands:
	"/ahoy on" (shows the window)
	"/ahoy off" (hides the window)
	"/ahoy reset" (moves the window back to the center, in case you lost it)
	"/ahoy scale #" (manually scale the window, replace # with a number, default is 1)

-- Version 1.04 --
-	Added option to hide collected items (Quests will be hidden if they were completed on the current character, Mounts need testing, I don't have any)
-	Fixed Scale Up button (thanks: RohenDar)
-	Fixed a ton of mismatches and added new items, added Kindleweb Spider category, thanks to oiFrango
-	Fixed missing creature id's (thanks: Aznamir)

-- Version 1.05 --
-	Added the option to send a message in instance chat that tells players to focus on whatever clasification is selected, and also its potential loot. It's on a 3 second cooldown to semi prevent spam.
-	Named Chests now also contribute towards the score and appear in the tooltip (thanks to oiFrango for providing all the data)
-	Added Shrine buffs and debuffs in their tooltips 

-- Version 1.06 --
-	Split the trolls into 3 categories: Witherbark, Icetusk and Sandfury since they have separate transmog sets. (thanks oiFrango)
	Some of the plundered weapons are listed for all 3 since they might share the loot table.
-	Added 70 missing items to the database, from source: https://docs.google.com/spreadsheets/d/1iGF5sWMl-f3Zlr18s5f7fUdVxC01QpZKKomYKsMb9KU/

-- Version 1.07 --
-	Added the option to show potential loot in the mouse over tooltip (it's off by default, go to settings to enable it)
-	Added a Database View window, can be accessed by clicking the scroll icon in the menu.
-	Updated the Help window to include all the new features.
	At the moment it just lists all known creature types and items that belon to them.
-	Added 3 missing Old God npc's (thanks oiFrango)
-	Moved all enemy faction NPC's under the PvP category, for database cleanliness sake

-- Version 1.07a --

-	Toggling off the tooltip details won't show info on shrines and chests now.
